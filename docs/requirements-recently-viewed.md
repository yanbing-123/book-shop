# 最近浏览功能

## 概述
在商品列表页顶部增加「最近浏览」模块，记录用户点击查看过的商品，方便快速回访。

## 功能需求

### 记录规则
- 用户通过「商品详情弹窗」查看商品时，自动记录该商品 ID
- 最新浏览的排在首位（unshift）
- 自动去重：同一商品重复浏览时，移到列表最前面
- 最多保留 10 条记录，超出则丢弃末尾

### 数据持久化
- 使用 `localStorage`，key 为 `book_recently_viewed`
- 每次记录变更立即写入

### 界面展示
- 位于主容器顶部、商品网格上方
- 区域标题：「🕐 最近浏览」+ 右侧「清空」按钮
- 横向可滚动卡片列表（`.rv-track`）
- 每张卡片显示：emoji、书名、作者、价格
- 点击卡片跳转到对应商品的详情弹窗
- 无记录时整块区域隐藏（`display: none`）

### 清空功能
- 点击「清空」按钮弹出确认框「确定要清空最近浏览记录吗？」
- 确认后清空所有记录并隐藏区域
- 清空后显示 Toast 提示「最近浏览记录已清空」

### 响应式
- 平板：卡片宽度 `110px`
- 手机（≤480px）：卡片宽度 `100px`，emoji 尺寸缩小

## 涉及的代码文件
- `index.html` — 最近浏览区域 DOM 结构
- `css/style.css` — 最近浏览样式（容器、卡片、滚动条、响应式）
- `js/app.js` — 最近浏览逻辑（记录、渲染、清空、localStorage 持久化）
