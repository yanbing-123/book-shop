# 📚 小型书籍购买系统 (book-shop)

基于 HTML + CSS + JavaScript 的纯前端书籍商品展示与购物系统。

## 功能特性

- 📚 **10种书籍**：红楼梦/三体/人类简史/小王子/活着/Python编程/资本论/十万个为什么/百年孤独/时间简史
- 🏷️ **分类筛选**：全部/文学/小说/科幻/科技/儿童/历史/社科（即时切换）
- 🛒 **购物车**：侧边抽屉、+/-数量、删除、清空购物车、实时总价、角标计数
- 💾 **数据持久化**：localStorage 独立存储 `book_stock` + `book_cart`
- 📦 **结算功能**：姓名/手机/地址验证 + 订单成功页（含书名+数量+小计）
- 🔒 **三层防超卖**：`addToCart()` / `updateQuantity()` / `submitOrder()` 全库存校验
- 📱 **响应式设计**：手机单列 / 平板双列 / PC四列自适应
- 🎨 **书香文艺主题**：暖米色背景 + 深棕色文字 + emoji📚图标 + Georgia衬线字体

## 本地运行

直接在浏览器中打开 `index.html` 即可运行，无需任何构建工具或服务器。

## 技术栈

- HTML5
- CSS3（Flexbox + Grid + CSS Variables + 暖米色书香主题）
- Vanilla JavaScript（ES6+）
- localStorage API
